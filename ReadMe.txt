CoreBlock's read me file.
This module contains the main simple blocks in Terasology.
Blocks that are in this module are including but not limited to:
Dirt, Stone, Water, Lava, Doors, Chest, Torches, Medal bars, Glass, Bricks, 
Planks, Rubber, Ice, Ladders, Cobble Stone, Ore blocks of different kinds,
Gravel, Grass, Snow, Plenty of different Woods and a bunch of Plants.
